Плагин "Loginza" (версия 1.0.0) для CMS LiveStreet
Работа плагина проверена на 0.4.2

= О плагине =
Плагин позволяет использовать для входа на сайт учетные записи провайдеров авторизации, таких как: Вконтакте, Google, Yandex, Mail.ru и д.р. Полный список поддерживаемых провайдеров указан на сайте http://loginza.ru. Механизм авторизации реализован на основе API сервиса Loginza.

= Возможности =
* Настраиваемый через конфиг набор и порядок расположения кнопок провайдеров в виджете Loginza. Вы можете задать список провайдеров, через которые хотите принимать авторизацию на сайт.
* Мультиязычный интерфейс виджета авторизации. Доступные языки: русский, украинский, английский и др. (полный список http://loginza.ru/api-overview). Можно использовать как авто выбор языковой версии виджета, так и отображение только заданной версии языка интерфейса.
* Заполнение профиля нового пользователя данными профиля учетной записи у провайдера (аватарка, ФИО, ник и д.р. доступные у провайдера данные).
* Прикрепление нескольких учетных записей провайдеров к одной учетной записи LiveStreet. Позволяет входить на сайт под одной учетной записью использую различные учетные записи провайдеров.
