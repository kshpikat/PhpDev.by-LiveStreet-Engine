ОПИСАНИЕ

Плагин заменяет стандартный простой редактор для LS на CKEditor. Для CKEditor можно использовать все доступные плагины и скины.

НАСТРОЙКА

Настройка плагина производится в файле config/config.php. Необходимо указать, какую библиотеку JavaScript использовать. На выбор - JQuery или MooTools
